## Turn long URLs into short links instantly

Practice project built with Go following the [roadmap.sh](https://roadmap.sh) challenge. A practical way to master the language while solving a real problem.

---

## What does GoTiny do?

**GoTiny** transforms long, impossible-to-remember URLs into short links you can share easily.

A minimalist link shortener that demonstrates how to build fast and efficient web services with Go.

---

## Key features

- **Instant shortening** — Convert URLs in milliseconds
- **Clean REST API** — Simple and predictable endpoints
- **Efficient storage** — Fast lookup for shortened links
- **Automatic redirects** — From short links to original destinations
- **Minimal dependencies** — Built primarily with Go's standard library

---

## Tech Stack

- **Go** — Fast and concurrent language for backend services
- **Simple architecture** — Straightforward design without unnecessary complexity
- **REST API** — Clear and standard interface

## Project purpose

Practice challenge from [roadmap.sh](https://roadmap.sh) to learn Go by building a real service. It allowed me to explore:

- HTTP handling in Go
- Data persistence
- API design
- Unique identifier generation
- Web service architecture
